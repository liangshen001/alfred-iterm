## @liangshen/alfred-iterm

*Alfred工作流，快速筛选使用ITerm Profiles功能*

*Alfred workflow for ITerm2*

### 运行环境

* Nodejs 14.x.x
* Alfred Powerpack
* Iterm2

### 安装

```
npm i -g @liangshen/alfred-iterm
```

### 使用方法

可使用 option + S 快捷键调出本工作流搜索框(或者调出alfred搜索框输入关键字 iterm)
